## 特性

- 安装 `forge` `fabric` `optifine` `quilt` 等模组加载器
- 监控游戏日志，可以指定游戏输出日志时执行的操作
- 读取、修改nbt文件
- 能够解析投影模组的文件、已安装的模组和资源包
- 指定文件下载源
- 跨平台，支持 `Windows` `GNU/Linux` `MacOS`
- 内置多线程下载，你可以根据自己的需要指定线程数
